# CLAUDE.md — PythonRefresher

See the [root CLAUDE.md](../CLAUDE.md) for repo-wide context.

Standalone Python practice scripts from the course's Python refresher section — variables,
strings, lists, dicts, sets/tuples, loops, functions, OOP (inheritance, polymorphism,
composition), imports, etc. Each topic folder holds independent scripts (often a lesson
file plus an `*Assignment.py`).

Not an application — no shared entry point, no dependencies beyond the standard library.
Run any script directly:

```bash
python "OOP/6. OOP/main.py"
```
